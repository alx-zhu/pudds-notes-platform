Option A
Stepped Wizard
Break the 9 metrics into 3 focused steps — one per category. Users complete each section before advancing. Reduces cognitive load by hiding what's not relevant right now.

Option B
Focus Mode — One at a Time
Rate one metric at a time with large, thumb-friendly buttons. Auto-advances after each selection. Perfect for hands-occupied lab use — minimal reading, maximum speed.

Option C
Segmented Row Controls
All 9 metrics shown in a clean scrollable list. Each row has a compact iOS-style segmented control that is larger and cleaner than the current dot buttons. Familiar, fast, and scannable.

Option D
Category Cards
Each sensory category gets its own visual card. Metrics are grouped inside. Large, clearly bordered buttons. The card header shows completion status at a glance, making it easy to see what's done.

Expandable Rows
