Design Decisions (v3)

  • Larger cards: 320px min-width. Up from 280px. More internal padding (18px vs 16px). Larger overall score (28px vs 24px). Taller bar gaps (6px vs 5px). The card breathes.
  • Formulation: abbreviations only, no percentages. "MC + GA" instead of "MC 12% / GA 0.4%". Sits on the same baseline as the trial number. The "+" separator is lighter weight. Much cleaner at a glance.
  • Score source shown as caption. "Thermomix · 3 Days" appears as an italic muted line below the score bars. This tells the user exactly which analysis log produced these scores without cluttering the score display itself.
  • Processing + flavor are plain muted text. No colored badges at all. "Benchtop · Vanilla" as simple dot-separated text in the footer. Eliminates the competing-colors problem entirely. The only vivid colors on the card are the 3 score bars.
  • Image overlay kept for context. The eval label overlay on the image thumbnail provides quick context when scanning photos, but it's the score-source caption below the bars that's the primary label for the score data.
  • Consistent color budget. Exactly 3 colors: emerald (taste), blue (texture), amber (color). Everything else is grayscale/neutral. The card feels calm and scannable.
  • Empty state preserves layout. Same overall + category rows with em-dash values and muted labels. Card height stays consistent across all cards.
